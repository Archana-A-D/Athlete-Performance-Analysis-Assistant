# AI-Powered Athlete Performance Analysis Assistant

## Project Overview
This project is an AI-powered Athlete Performance Analysis Assistant developed using IBM watsonx Orchestrate, IBM Granite Models, LangFlow, and Retrieval-Augmented Generation (RAG). The system analyzes athlete performance, identifies strengths and weaknesses, and provides personalized training recommendations and coaching insights.

## Features
- Athlete Performance Analysis
- Performance Trend Analysis
- Personalized Training Recommendations
- Coaching Reports
- Multi-Agent Architecture
- RAG-based Knowledge Retrieval
- Interactive Dashboard

## Technology Stack
- IBM watsonx Orchestrate
- IBM Granite Models
- LangFlow
- IBM watsonx.ai
- IBM Cloud
- RAG
- FAISS / ChromaDB
